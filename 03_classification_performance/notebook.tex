    \hypertarget{classification_pt1}{%
\section{03\_Classification\_pt1}\label{classification_pt1}}

In the previous tutorials, we've seen machine learning takes the problem
form:

\begin{quote}
\texttt{Given} - x1, x2, x3\ldots{} (features)\\
- y (target)
\end{quote}

\begin{quote}
\texttt{Our\ wish} - Find a good predictor/profiler (or ``function'')
that help us predict target, given the features
\end{quote}

From now on, we would discover one of the the most fundamental task in
machine learning: classification.

\textbf{WARING: This tutorial is focusing on concepts. I strongly
recommend you raise any question you have in this section.}

    \hypertarget{p1-what-is-classification}{%
\section{P1: What is classification?}\label{p1-what-is-classification}}

X = categorical or numerical Y = ``group'' or ``class'' membership of
record

For a function that help us predict y(target) given x1,x2,
x3\ldots{}(features), we usually call it a \textbf{classifier}

\begin{itemize}
\tightlist
\item
  Goal: To explain or to predict?

  \begin{itemize}
  \tightlist
  \item
    Profiling (=Explaining): What factors (X1, X2,\ldots{}) affect
    customer satisfaction (Y)?

    \begin{itemize}
    \tightlist
    \item
      Purpose: Retention strategies, Acquisition strategies, Policy
      changes
    \item
      Data: Customer demographics, Info on Customer calls, Billing
      history
    \end{itemize}
  \item
    Classification (=Prediction):Predict Y (class) for new records,
    based on their feature information (X1, X2,\ldots{})\\
    \texttt{can\ you\ think\ of\ a\ predictive\ goal\ for\ the\ predictive\ example\ using\ the\ same\ data?}
  \item
    Ranking: a special predictive goal, will discuss later
  \end{itemize}
\end{itemize}
